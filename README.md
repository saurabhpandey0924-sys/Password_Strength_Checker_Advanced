# 🔐 Password Strength Checker

A modern Password Strength Checker built using HTML, CSS, and JavaScript.

## 🚀 Features

- Real-time password strength analysis
- Password entropy calculation
- Password risk score (0-100)
- Crack time estimation
- Validation checks:
  - Minimum 8 characters
  - Uppercase letters
  - Numbers
  - Special symbols
  - Bonus for 12+ characters
  - Common leaked password detection
- Interactive UI with terminal-inspired design
- Password visibility toggle
- Dynamic strength meter and score ring

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
